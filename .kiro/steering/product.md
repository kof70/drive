# Product Overview

Local Collaborative Workspace is a real-time collaborative canvas application for local networks. Users can share files, notes, and organize content visually on an infinite canvas without requiring internet connectivity.

## Core Features

- Visual interactive canvas with drag-and-drop
- Real-time multi-user synchronization via WebSocket
- Cross-platform support (desktop, mobile, tablet)
- Local network only - no internet dependency
- Canvas elements: notes, folders, files, images
- Zoom, pan, and multi-select capabilities

## Target Use Case

Designed for trusted local network environments (home, office) where teams need to collaborate and share content without cloud dependencies or internet access.

## Security Model

**Important**: No authentication or encryption. Intended for trusted local networks only. Never expose to the internet.
